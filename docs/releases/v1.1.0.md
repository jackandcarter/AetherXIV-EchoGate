# MeteorXIV Core + Echo Gate v1.1.0

This is the first packaged release pass for MeteorXIV Core and Echo Gate.

## Included

- Echo Gate launcher builds for Linux, macOS, and Windows targets.
- A MeteorXIV Server Core package with built Lobby, Map, and World server outputs.
- SQL, PHP launcher services, configs, scripts, setup helpers, and documentation in the server core package.
- Local MariaDB defaults through `.env.defaults`.
- Echo Gate account creation, login, news/status, runtime selection, client validation, patch validation, and launch diagnostics.

## Important Notes

- macOS apps are ad-hoc signed but not Apple-notarized, so Gatekeeper may ask for manual approval.
- Steam Deck support is experimental and should use the Linux x64 artifact as a starting point.
- The server core package is not a one-click hosted server. You still need MariaDB/PHP and local setup.
- No release asset includes FFXIV client files, patch files, patch torrents, Wine runtimes, or `Data/staticactors.bin`.
- Prepare `Data/staticactors.bin` locally from your own FFXIV 1.x client folder before real Map Server playtests.

## Recommended Setup

Start with `docs/RELEASES.md`, then use the platform guide for your machine.
